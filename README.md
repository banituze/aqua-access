# AquaAccess

Innovative, sustainable, community-centered clean water solutions across Africa.

## Project Overview

This website represents the AquaAccess initiative, a mission-focused organization aligned with ALCHE's Global Challenges and Global Opportunities (GCGO). The website communicates the mission to enhance access to clean water across Africa by creating innovative and sustainable water solutions that are both affordable and accessible.

## Features

- **Home Page :** Clear mission statement, vision, and focus areas
- **Mission Page :** Detailed alignment with GCGO, objectives, and media content
- **About Page :** Bio, technical and soft skills
- **Impact Page :** Planned and projected impact
- **Contact Page :** Interactive contact form with validation

## Getting Started

1. Clone the repository :
```
git clone https://github.com/banituze/aqua-access.git
```

2. Open the website :
- Simply open `index.html` in your web browser
- Or use a local development server:
```
# Using Python 3
python -m http.server 8000
     
# Using Node.js (with http-server)
npx http-server
```

3. Navigate to `http://localhost:8000` (or the port specified by your server)

## Technologies Used

- **CSS3 :** Custom styling with layout, reset, and style sheets
